# Return By Death
This mod was created by the request of  DogeTOM on fabric project discord server <br>
Thanks to [ThePotatoArchivist](https://github.com/ThePotatoArchivist/) for helping in creation of this mod through [save state](https://github.com/ThePotatoArchivist/SaveState) mod

[discord thread](https://discord.com/channels/507304429255393322/1518333648389279927)


This mod added the return by death ability from Re:Zero

#### There are multiple ways to save checkpoints:
    By sleeping on a bed 
    By completing an advancement 
    or if 20 minutes ( one Minecraft day passes) 

#### there are two mods, instant return by death or normal
##### by default, it uses normal mode:
    in this mode when you die, you will get the death screen and you can choose to return by death or exit the game
##### The Instant Return Mode:
    enable immediate_respawn game rule using
    `/gamerule immediate_respawn true`

Now if you die you instantly get returned to death.

There are sound effects for each return.
